Emerson Ipsum

What is it?

Emerson Ipsum provides the user with filler text, but with a distinction: the text is generated by a recurrent neural network trained on the written work of 19th-century American poet and philosopher Ralph Waldo Emerson.

Due to the nature of RNNs, the output text is pseudo-grammatical, often sublime, sometimes neologistic, and totally preposterous. In other words, it's all the best things language can be.

JSON API

With the JSON API, you can incorporate Emerson Ipsum into your own projects. The endpoint for the API is...

https://mikefooks.com/emerson/api

Parameters

name description default
paras The number of paragraphs you'd like to generate 1
sentences The number of sentences per paragraph. 4
headings generate optional headings, to mock up headlines, titles, things of that nature. takes an integer. 0
format The format for the response. You can choose from "json", "html", or "text" json
