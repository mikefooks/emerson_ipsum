Emerson Ipsum

What is it?

Emerson Ipsum provides the user with dummy text, but with a distinction: this text is generated by a recurrent neural network trained on the written work of 19th-century American poet and philosopher Ralph Waldo Emerson.

Due to the nature of RNNs (and to some extent Emerson's prose), the output text is often sublime, sometimes neologistic, and strains any traditional definition of comprehensibility.

Test it out

Use the sliders and the generate button to get some free dummy text.

API

You can incorporate Emerson Ipsum into your own projects using the API. The endpoint is 

https://mikefooks.com/emerson/api

Parameters

name description default
paras The number of paragraphs you'd like to generate 1
sentences The number of sentences per paragraph. 4
headings generate an optional number of headings, to mock up headlines, titles, things like that. takes an integer. 0

Examples

  • https://mikefooks.com/emerson/api?headings=3&sentences=4
